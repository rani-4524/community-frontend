# CommunityHub

A MERN Stack community management platform where users can join communities, RSVP to events, and hosts can create and manage communities and events.

## Live Demo
https://communityhub-live.vercel.app

## Frontend Repository
https://github.com/rani-4524/community-frontend

## Backend Repository
https://github.com/rani-4524/community-backend

## Features
- User Authentication (JWT)
- Host Dashboard
- Community Creation & Management
- Event Creation & RSVP
- Profile Picture Upload
- Responsive Design
